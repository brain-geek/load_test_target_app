== Welcome to Load Test target app

This app was intended to use with Toad - website load testing project.

== Setting up empty VM in ubuntu

    sudo vmbuilder kvm ubuntu --suite precise --flavour virtual --arch i386 -o --libvirt qemu:///system --ip 192.168.0.201 --hostname myvm --user brain --name user --pass password --addpkg unattended-upgrades --addpkg openssh-server
